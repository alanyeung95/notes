## check windows reboot history

1. open windows event viewer
2. open `Windows Logs` tab on the left hand side, then `System`
3. select `Filter Current Log...` on the right hand side
4. filter event with event id

ref: https://manage.accuwebhosting.com/knowledgebase/3897/How-to-check-shutdown-and-reboot-logs-in-Windows-servers.html
